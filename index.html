Projects I've done


Steganography


Stack: Python, Qt


A graphical user interface built upon Python and the QT framework. The gui let's the user conceal an image within another image, barely modifying the holder image, visually. The gui uses smart and easy bit level operations to complete the steganography of any image within a matter of seconds.

Intelligent Text Editor


Stack: java se


This intelligent text editor first learns a large dictionary(can be provided by the user), the concept of edit distance and the trie data structure, made using linked lists and provides the user with an intelligent text editor featuring autocorrection and autocompletion. The text editor also features a Flesch index calculator and a Markov text generator.

Personal Website


Stack: html, css, and JavaScript


An interactive website using clean and simple material design. It is built upon html, css, and the JavaScript, jQuery framework.

At811


Stack: Android Studio


A prototype for an augmented reality android application, that eliminates for the user, the need to call 811 before he undertakes any digging project, by showing him the location of gas, water, or electricity lines, running beneath the ground.

Xpense


Stack: html, css, and JavaScript


A web application geared towards college students, that promotes investing on top of saving and helps college students make smart investment decisions. Features clean and simple material design and is mainly built upon the Nessie api by Capital One.

Appointment Scheduler


Stack: html, css, JavaScript, asp.net mvc4 framework, mysql


Another web application built using the asp.net mvc4 framework and the visual studio entity framework for easy database access. The application allows a group of people, in this case students, counselors, and administrators to schedule, approve, and manage appointments respectively.
